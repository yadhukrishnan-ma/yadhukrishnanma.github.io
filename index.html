Developed a 2D SLAM pipeline using LiDAR sensor data with the SLAM Toolbox. Generated occupancy grid maps, implemented pose estimation and localization, and validated loop closure accuracy in multiple simulated environments.

PRJ-004 // HARDWARE + ROS 2
ESP32 → ROS 2 · 5-DOF IMU Robotic Arm

Bridged physical hardware to ROS 2 in real time. Two MPU-6500 IMUs on separate I2C buses feed an ESP32 running an on-board Madgwick filter to fuse raw accelerometer + gyroscope data into quaternions. Streamed over 921600-baud serial to a Python ROS 2 bridge, which publishes sensor_msgs/Imu. A second node converts quaternions to joint angles and drives a live 5-DOF URDF arm in RViz2.

ESP32 MPU-6500 IMU Fusion Madgwick Filter ROS 2 Serial Bridge JointState URDF RViz2
