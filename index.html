PRJ-003 // PERCEPTION
LiDAR SLAM Mapping

Developed a 2D SLAM pipeline using LiDAR sensor data with the SLAM Toolbox. Generated occupancy grid maps, implemented pose estimation and localization, and validated loop closure accuracy in multiple simulated environments.

SLAM Toolbox LiDAR Occupancy Grid Localization
PRJ-004 // HARDWARE + ROS 2
ESP32 → ROS 2 · 5-DOF IMU Robotic Arm

Bridged physical hardware to ROS 2 in real time. Two MPU-6500 IMUs on separate I2C buses feed an ESP32 running an on-board Madgwick filter to fuse raw accelerometer + gyroscope data into quaternions. Streamed over 921600-baud serial to a Python ROS 2 bridge, which publishes sensor_msgs/Imu. A second node converts quaternions to joint angles and drives a live 5-DOF URDF arm in RViz2.

ESP32 MPU-6500 IMU Fusion Madgwick Filter ROS 2 Serial Bridge JointState URDF RViz2
PRJ-005 // PERCEPTION
Precision 6D Pose Calibration (SO-101)

Developed a high-precision 6D pose tracking pipeline for the SO-101 robot arm. Implemented adaptive thresholding to handle reflective overhead light noise, multi-face anchor sets to track marker constellations at steep tilts (up to 66°), and a guided interactive calibration GUI.

ROS 2 solvePnP RANSAC OpenCV RViz2 Calibration
PRJ-006 // PERCEPTION + AI
Industrial 6D Pose via DenseFusion

Designed a modular 6D object pose estimation pipeline. Combines YOLOv8 instance segmentation masks with a DenseFusion model to fuse RGB appearance (ResNet-18) and 3D geometric point clouds (PointNet) for per-point pose voting and iterative refinement.

DenseFusion YOLOv8 PyTorch Open3D CUDA
PRJ-007 // PERCEPTION + HARDWARE
ArUco Marker 6-DOF Ground Truth Tracker

Created a ROS 2 node using a laptop camera to track 3.7cm ArUco markers on a robotic arm. Uses solvePnP with IPPE_SQUARE, applies a 5-frame moving average filter, and publishes pose accuracy comparisons against Forward Kinematics wrist coordinates in real time.

ArUco solvePnP PoseStamped Sensor Calibration ROS 2
PRJ-008 // NAVIGATION + SIMULATION
Autonomous TurtleBot Stack

Engineered a complete simulation and localization stack for a custom 12.6cm x 12.6cm TurtleBot. Developed a detailed URDF description and configured custom Nav2 costmaps and footprints for navigation through tight spaces.

ROS 2 Cartographer Nav2 URDF Costmaps
PRJ-009 // HARDWARE + DEPLOYMENT
TortoiseBot Remote Deployment & Diagnostics

Configured netplan and ROS network parameters for Raspberry Pi 4-based TortoiseBot systems. Developed a remote python diagnostic suite verifying motor drivers, compressed camera feeds, and LiDAR scanning topics.

Raspberry Pi Netplan SSH Diagnostics L298N
PRJ-010 // PERCEPTION + CONTROL
Webcam-Based Arm Motion Capture
